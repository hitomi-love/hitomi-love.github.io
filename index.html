WHAT HITOMI IS

Built like a playful overlay. Behaves like a real operator.

01

Floats above your screen

Hitomi lives as a draggable hedgehog overlay, so you can call on her without leaving the app you are already using.

02

Browses inside the app

She can open a compact browser window for lookups, step-by-step navigation, and lightweight web tasks.

03

Reaches Android terminal commands

The advanced bridge can reach Android terminal commands directly through Termux when you deliberately turn that mode on.

04

Logs in with a Solana wallet

Users can log in with a Solana wallet, unlocking a wallet-aware mode for people who already live onchain instead of treating crypto as a separate app universe.

05

Two modes, one companion

Hitomi is friendly by default for everyday Android users, but she still has enough teeth for power users who want browsing, voice, Solana-aware monitoring, and terminal-connected workflows in one tiny assistant.

SOLANA MODE

Wallet login, onchain monitoring, and a current free usage path.

Hitomi can authenticate through a Solana wallet flow, inspect wallet balance and recent transactions, and use that onchain context as part of the conversation instead of treating crypto like a disconnected side topic.

  • Wallet-auth login
  • Balance checks
  • Recent transaction monitoring
  • Onchain-aware chat

Wallet-authenticated users currently receive 12,000 LLM tokens per day. That is a live perk right now, not a forever guarantee.
